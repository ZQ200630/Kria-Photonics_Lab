Lock Status-

Laser On enters static hold using the static setpoint below. Laser Off ramps both channels to zero. Emergency Stop disables immediately.

Static Setpoint

Set Static updates the static target. If the laser is already on, it leaves scan mode and ramps to this static target without turning the laser off.

Fine Scan

Scan drives CH1 start to stop during the saved forward frame, then ramps back to start without saving the return. Completed scans hold CH1 at start and keep the laser enabled.
